WP Ultimate CSV Importer Pro

WP Ultimate CSV Importer Pro is a WordPress plugin that enables to import and export files. It supports CSV, XML, text and zip files. The files are imported in very few clicks, it can also be scheduled for later import. The plugin supports various third party eCommerce, SEO, custom posts and custom fields plugins. It also supports the WPML plugin. It helps you create a wonderful WooCommerce site. It also supports few WooCommerce add-on such as WooCommerce chained products, WooCommerce pre orders, WooCommerce retailer, WooCommerce products and WooCommerce returns and warranty.

Feature highlights

  • WPML plugin support.
  • Supports shortcode for both inline and featured images import.
  • Supports ACF fields along with repeater fields, WooCommerce addons and custom attribute feature.
  • Enables to schedule the file for later import. It also enables to update the existing files.
  • File manager to manage the imported files.
  • Supports event manager plugin(beta) and dropshipping (beta).
  • WP eCommerce custom fields multi check box support.

WP Ultimate CSV Importer

WP Ultimate CSV Importer is the free version of the plugin. It supports few third party plugins such as eShop, Custom Post Type UI and All in one SEO plugin. It allows to import any number of records in a single import. The size of imported file is based on the server configurations. The server configurations can be viewed in the settings section of the plugin.

User interface with flexible jquery plugin and bootstrap makes CSV import more simple, easy and fast with better user experience. With improved multi site, buddy press compatibility (except user import module) works well for most of the best WordPress themes available online. Now manage multi site & buddy press, Amazon data import, affiliates blog, product inventory, eCommerce store, multi author blog, automatic news import, photo blog etc periodically with simple click import.

Feature highlights

  • Imports inline images with shortcodes.
  • Featured image import with external URL, if the theme supports.
  • WordPress core custom fields can be registered on the flow of import.
  • It can be activated in multisite
  • Multi category and multi tag import.
  • Enables to imports selective records from the file.
  • Exports all the imported files of different modules with filters.

File import procedure

Importer has four simple sections to import a file. It imports bulk data in a few minutes of upload.

The file upload section enables to import file in 4 different methods. They are browse from computer, FTP upload, external URL link and upload from server. These methods uploads bulk records in considerably few minutes.

The mapping section enables to map the WP fields with the CSV headers. It has a easy flow of import with custom field registration on the flow, (i.e) it enables to register the WordPress fields, ACF, Types and Pods custom fields on the mapping section of the import process. It also enables to save the current mapping as a template for later update or scheduling of files.

The summary section enables to know the number of records along with number of mapped and unmapped fields.

Finally the import section enables to detect duplicate title and content, it also provides selective record import. The image in the shortcodes are uploaded under image handling.

Featured Image theme support

If the current theme of your site doesn't support featured image, add the below code where the imported featured image need to be visible.

<?php add_theme_support( 'post-thumbnails' ); ?>

E.g. wp-contents/themes/header.php

Important notes

  • CSV files should be in UTF-8 format.
  • User import in multisite is not possible
